## Carregando ambiente virtual

Criando o ambiente virtual:

``` python
virtualenv extrator-placas
```

Ativando o ambiente virtual:

``` python
source extrator-placas/bin/activate
```

Desativando o ambiente virtual:

``` python
deactivate
```
